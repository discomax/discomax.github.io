Weather vs Latitude

In this project, I used Python, APIs, and JSON traversals to answer a fundamental question: "How does proximity to the equator affect the weather?" Does the data support the idea that it simply gets hotter as we get closer?

To do this, I created a Python script to visualize the weather of 500+ random cities across the world with varyious distances from the equator. I obtained the cities and weather with the citipy Python library, the OpenWeatherMap API.

Comparisons Used:

  • Temperature (F) vs. Latitude
  • Humidity (%) vs. Latitude
  • Cloudiness (%) vs. Latitude
  • Wind Speed (mph) vs. Latitude
